# Servlet DVD App

Spring MVC + Servlet/JSP 기반 DVD 관리 웹앱.

**Tech:** Java, Servlet, JSP, Maven

## Structure

```
src/main/webapp/
├── WEB-INF/views/    # JSP 뷰 (form, search, add, result)
├── static/           # JS 파일
└── web.xml
```

## Features

- DVD / Producer / Publisher 조회 및 등록
- Servlet → JSP 뷰 렌더링
