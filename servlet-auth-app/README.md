# Servlet Auth App

Spring MVC + Servlet/JSP 기반 로그인/회원가입 웹앱.

**Tech:** Java, Servlet, JSP, Maven

## Structure

```
src/main/webapp/
├── WEB-INF/
│   ├── views/      # login, register, mypage, index JSP
│   ├── include/    # header, common_css
│   └── web.xml
└── static/         # CSS, JS
```

## Features

- 회원가입 / 로그인 / 마이페이지
- JSP include 구조 (공통 헤더/CSS)
