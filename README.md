## Tech Stack

- **Java** — Maven, Java 17+
- **Frontend** — HTML5, CSS3, Vanilla JS
- **React** — CRA, React Hooks, Axios, React Query, Recoil
- **Database** — MySQL, JDBC, H2
- **Backend** — Spring Boot, MyBatis, Spring Security, JWT, OAuth2
