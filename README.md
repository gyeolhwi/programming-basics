## Tech Stack

- **Java** — Maven, Java 17+
- **Frontend** — HTML5, CSS3, Vanilla JS
- **React** — CRA, React Hooks, Axios
- **Database** — MySQL, JDBC
- **Backend** — Spring Boot, MyBatis, REST API
